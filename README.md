# TypeScript Learn 

A structured and hands-on repository to **learn TypeScript from basics to intermediate concepts**, built step-by-step through daily commits.  
This repository focuses on **core TypeScript fundamentals, OOP concepts, generics, interfaces, and real-world API typing using Axios**.

---

##  Repository Objective

This repository is created to:
- Understand **TypeScript fundamentals** clearly
- Learn **type safety** and best practices
- Practice **real-world typing scenarios**
- Build a strong foundation for **frontend and backend TypeScript projects**
